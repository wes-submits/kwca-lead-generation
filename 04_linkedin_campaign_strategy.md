## Campaign KPIs & Optimization

### Success Metrics

| Metric | Target | Review Frequency |
|--------|--------|------------------|
| CTR (Sponsored Content) | 0.4-0.8% | Daily |
| CTR (Message Ads) | 8-15% | Daily |
| Landing Page CVR | 15-25% | Weekly |
| CPL (Cost Per Lead) | $95-165 | Weekly |
| SQL Rate | 25-35% | Monthly |
| Demo → Close Rate | 20-30% | Monthly |

### Optimization Triggers

| Condition | Action |
|-----------|--------|
| CTR < 0.3% (3+ days) | Refresh creative, test new headline |
| CPC > $30 | Narrow audience or adjust bid strategy |
| Landing CVR < 10% | A/B test headline, reduce form fields |
| Message Ad CTR < 5% | Rewrite opening line, test new CTA |
| High CPL (> $200) | Pause lowest-performing ad set, reallocate budget |

### Budget Allocation (Weekly Review)

```
Week 1: Equal split ($125/day per ad set)
Week 2+: Reallocate to best performers
  - If Message Ads winning: $200/day Message, $100/day Sponsored
  - If Sponsored Content winning: $300/day Sponsored, $66/day Message
```
